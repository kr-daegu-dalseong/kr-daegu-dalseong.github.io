# 달성군 · 대구광역시

> 달성군 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-daegu-dalseong.github.io/**

달성군은 대구광역시 유일의 군(郡)이면서도 전국 군 지역 가운데 인구가 가장 많은, 도시와 농촌의 얼굴을 함께 지닌 곳입니다. 낙동강이 서쪽 경계를 감싸 흐르고 남동쪽으로 참꽃(진달래)으로 이름난 비슬산이 우뚝 솟아, 강과 산이 군의 골격을 이룹니다. 화원읍·다사읍처럼 대구 도심과 맞붙어 도시철도가 연결된 생활권이 있는가 하면, 유가읍·현풍읍 일대에는 대구테크노폴리스와 대구경북과학기술원(DGIST)이 들어선 연구·산업 신도시가 새로 자라났습니다. 우리나라에 처음 피아노가 들어온 사문진 나루터와 유네스코 세계유산 도동서원이 나란히 자리해 근대사와 유교 전통을 동시에 품고 있습니다. 봄이면 비슬산 능선을 뒤덮는 분홍빛 참꽃 군락이 군 전체를 상징하는 풍경으로 손꼽힙니다.
